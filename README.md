# 🚀 Task Tracker (MERN Stack)

A responsive and modern **Task Tracker Web Application** built using the **MERN Stack**. This application helps users efficiently manage their daily tasks with complete CRUD functionality and a clean user interface.

---

## 📸 Project Preview

> Add screenshots after deployment.

## ✨ Features

- ✅ Create Tasks
- ✅ View All Tasks
- ✅ Update Existing Tasks
- ✅ Delete Tasks
- ✅ Search Tasks
- ✅ Task Statistics (Total, Pending, Completed)
- ✅ Delete Confirmation
- ✅ Responsive UI
- ✅ Form Validation
- ✅ Dynamic Updates (Without Page Refresh)
- ✅ MongoDB Integration

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/task-tracker-mern.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## 🌐 API Endpoints

| Method | Endpoint | Description |
|---------|----------|-------------|
| GET | `/api/tasks` | Fetch all tasks |
| POST | `/api/tasks` | Create a new task |
| PUT | `/api/tasks/:id` | Update a task |
| DELETE | `/api/tasks/:id` | Delete a task |
